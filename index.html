RAKTIM-MASTER-PDF-UTILITY

I need to compare tons of PDF files for multiple purposes. I could not find any good FREE library which is working out of the box to compare the PDF files. I did not want just Text compare & I was looking for something which can compare PDFs pixel by pixel to find all the differences. Libraries which can do are NOT FREE. So, I have come up with a simple JAVA library (using apache-pdf-box – Licensed under the Apache License, Version 2.0) which can compare given PDF documents in Text/Image mode & highlight the differences, extract images from the PDF documents, save the PDF pages as images etc

USAGE

I have used pdfbox-tools api to develop this utility.Advanced java is the main usp of this enhancement .Lots of advanced java concepts like enums , collection framework, String , reflection API have been the key components of this utility. Anyone can integrate this into thier framework by providing the following dependencies in pom.xml or by attaching the .jar file inside their classpath

====Following are the usage of this utility====

1)To get page count from a particualr pdf

2)To get page content as plain text from any page/pages

3)To extract attached images from PDF for multiple pages

4)To store PDF pages as images for any page/pages

5)To compare PDF files in text mode (faster – But it does not compare the format, images etc in the PDF)

6)To extract attached images from PDF for page-2 & 3To compare PDF files in Visual mode (slower – compares PDF documents pixel by pixel – highlights pdf difference & store the result as image)
